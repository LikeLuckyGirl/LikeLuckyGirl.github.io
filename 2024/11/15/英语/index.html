第一章   动词时态

英语的时态共有16种，专升本常考的只有10种，并且重点测试“完成态”。（包括：现在完成时，过去完成时，将来完成时）
主动 被动
一般现在 do，does，am,is,are Am done,is done,are done
一般过去 did,was,were was done,were done
一般将来 will do,be going to do will be done,am/is/are going to be done
一般过去将来时 would do,was/were going to do would be done,was/were going to be done
现在进行 am/is/are doing am/is/are being done
过去进行 was/were doing was/were being done
将来进行 will be doing will be being done
现在完成 have/has done have/has been done
过去完成 had done had been done
将来完成 will have done will have been done

注释：

Do代表所有动词的“原型”
Dose代表所有动词的“第三人称单数形式”（-s -es -ies）
Am/is/are的动词原型：be；（含义：是 成为）
Done代表所有动词的“过去分词”（-d -ed -ied）
Did代表所有所有动词的“过去式”（-d -ed -ied)
Was/were的动词原型：be
Doing代表动词的ing形式

被动结构：be+done 同时满足（背诵）改变被动语态的时态 改be(背诵)

(一)一般现在时（共三个考点）

定义：表示通常性、规律性、习惯性、真理性的状态或者动作有时间规律发生的事件的一种时间状态。
例如：I get up erly this week. 我这种通常早起。
    一般现在
例如：大自然的客观规律
The moon runs around the earty
    一般现在

考点一：一般现在时常见的时间状语（路标词）：always 总是,usually 通常,often 经常,sometimes 有时,every day 每天,every week 每周,on Mondays 在每周一 等

扩展：
     Next      下一个
     Next week 下周
     Next year 明年
     Next boys ~~下一些男孩 接下来的男孩们 下一批男孩~~ 后来的男孩们 即将到来的男孩们

e.g. Tom often gets/will get/got into the traps set by Jerry.(语法改错)
解析:找路标-定考点-出答案-翻译核实。（背诵）

Often   考点：一般现在。
Gets        does形式      一般现在；
Will get    will do形式   一般将来；
Got         did形式       一般过去；

Set v. 建立
Set upone’smindto do sth建立某人的想法去做某事（决定去做某事）
TV set 电视设备
Settle   定居
Settler   设定者 定居者

Minute   分钟
例如：he is not tall.（他是不高的） He is minute.（他是矮的）
（注意：此时此景的含义 > 我们头脑里的那个含义）

直击考点：
Now my father __ C __ his bike to work every day instead of driving.
A.ride   B.rode   C.rides   D.will ride
解析：
Now   考点：现在进行 am/is/are doing；
A   do形式   一般现在
B   did形式   一般过去   ride-rode-tidden 骑
C   does形式   一般现在
D   will do形式   一般将来
剩下两个选项   找区别。（背诵）
扩展：
Instead   adv.（副词） 代替   +不东西
Instead of…   +东西
Replace   v. 代替
例如：We will 情态动词 __ replace __ the position of him.
例如：Tom will do it __ instead of __ me.

考点二：在时间条件状语从句中，用一般现在时表示将来时（主将从现   含义：主句用一般将来时间，从句用一般现在时；从句用一般现在表示一般将来。）。

常用引导词(引出状语从句）：

  • 表示时间含义的：when 当……时候（考点：主将从现 时态一致）,until 直到……为止,after 在……之后,before,as soon as 一……就……,once 一旦,the moment ……的时候,the minute ……的时候,the day ……的那天   等。
    e.g.Once he gets/will get/get/got on the bus later 从句,he will not have 一般将来 to worry.
    解析：
    找路标-
    Once   一旦   时间引导词   考点： 类似 when
    Later   一会儿   考点：一般将来。
    Get on 上车
    Get off 下车
    Get up 起床
    例如：You should get the tent 帐篷 up now.It will rain 下雨.
    支起帐篷
  • 表示条件的：if 如果 考点：虚拟语气 主将从现在 时态一致。,unless(如果不   除非 = if……not),provided(假设,如果),as long as(如果,只要)等。
    e.g.If he accepts 一般现在 the job,he will get/gets more money soon.
    解析：找路标-
    If 如果   考点：虚拟语气 主将从现 时态一致。（背诵）
    虚拟语气判断：
    1. if 条件下
    2. 动词的形式是”过去的形式“
    3. 同时满足

过去的形式：一般过去，过去进行，一般过去将来，过去完成。
三个引导词：when   once   if;
More 更多的
例如：I need more water.   Adj.(形容词)
形容词修饰：名词或代词。（背诵）
Many/much–more–most   多
直击考点：
If 如果 you __ A __ your homework 从句,you can go out to play football.
A.finish    B.will finish    C.are finishing    D.finished
解析：找路标-
Can   could 过去式
剩下两个选项找区别。（背诵）
